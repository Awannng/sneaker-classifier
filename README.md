# Sneaker Classifier
<p>A web app that utilizes a custom-trained model to recognize shoe brands from uploaded images.</p>
<p>The brands includes: Adidas, Asics, Converse, New_Balance, Nike, Puma, Reebok, Skechers, Vans</p>
